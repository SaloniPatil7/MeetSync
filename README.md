# 🚀 MeetSync - Video Conferencing App

## 🌐 Live Demo

🔗 https://meetsync-n518.onrender.com/

---

## 📌 Overview

MeetSync is a real-time video conferencing web application that allows users to connect, communicate, and collaborate seamlessly. It provides features like video calling, screen sharing, and meeting history tracking, similar to modern platforms like Zoom.

---

## ✨ Features

* 🔐 Secure user authentication (JWT-based)
* 📹 Real-time video calling
* 🎤 Microphone control (mute/unmute)
* 🎥 Camera control (on/off)
* 🖥️ Screen sharing functionality
* 🕓 Meeting history tracking
* 📱 Responsive UI for all devices

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/SaloniPatil7/MeetSync.git
cd MeetSync
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=8080
MONGO_URI=your_mongodb_connection_string
```

Run backend:

```bash
node app.js
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
```

Run frontend:

```bash
npm start
```
